> Public subnet 과 Private subnet 을 포함한 VPC 를 생성 합니다.

![Success](../../terraform/images/terraform_vpc_ach.png)

> Terraform 명령으로 생성 합니다.

```
cd terraform-env-workshop/vpc

terraform init
terraform plan
terraform apply
```
